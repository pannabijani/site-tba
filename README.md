# PaññāyaBījāni

PaññāyaBījāni est un site dédié à l'étude des Textes Bouddhistes Anciens, en particulier des suttas (discours doctrinaux du Bouddha et de ses premiers disciples). Il contient de nouvelles traductions françaises des sutta par Noé Ismet. Il contient également diverses informations de références sur les Textes Bouddhistes Anciens.
